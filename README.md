# SLIIT Student 360 Demo

⚠️ **DISCLAIMER**: This is an educational demonstration project only. This is NOT affiliated with or endorsed by SLIIT. This should only be used for educational purposes to demonstrate UI/UX design and React development skills.

## About

This is a React-based replica of the SLIIT Student 360 login interface created for educational purposes to demonstrate:
- React component design
- Form handling
- Responsive design
- CSS styling techniques

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Technologies Used

- React 18
- CSS3
- HTML5

## Features

- Responsive login interface
- Modern UI design
- Form validation
- Hover effects and animations

## Note

This is a demonstration project only. No actual authentication or data submission occurs. All login attempts will show an alert message indicating this is a demo.
